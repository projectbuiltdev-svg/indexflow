   MIN_EXTERNAL_LINKS (1)
9. Image presence: img tag count > 0
10. Alt text check: ALL img tags have non-empty alt attribute
11. Schema presence: script[type="application/ld+json"] → 
    JSON.parse succeeds
12. Location name check: first paragraph text → case-insensitive
