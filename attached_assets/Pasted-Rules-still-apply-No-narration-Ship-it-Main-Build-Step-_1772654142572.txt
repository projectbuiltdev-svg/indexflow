Register: app.use('/api/we/comments', weAuth where

COMMENT ROUTES:

POST /api/we/comments/:token/add
- NO auth required — public endpoint for clients
- Validates preview token exists and not expired
  in we_deployments
- Resolves projectId and venueId from token
- Body: { elementSelector, xPercent, yPercent,
  comment, clientName, clientEmail? }
- elementSelector: CSS selector of clicked element
- xPercent/yPercent: position as % of viewport
  for pin placement
- Stores comment in we_form_submissions with
  formName: 'client_preview_comment'
- Returns { commentId, pinNumber }
- Emails agency via workspace SMTP if configured:
  Subject: 'New comment on [project name]'
  Body: client name, comment text, page, timestamp
- Rate limit: 20 comments per IP per hour

GET /api/we/comments/:token
- NO auth required — for client preview page
- Returns all unresolved comments for this token's project
- Returns { comments[], total }

GET /api/we/comments/project/:projectId
- Requires auth + venueId
- Returns all comments for project
- Supports ?resolved=true/false filter
- Returns { comments[], total }

PATCH /api/we/comments/:commentId/resolve
- Requires auth + venueId
- Sets resolved: true, resolvedAt, resolvedBy
- Logs to we_audit_log

POST /api/we/comments/:commentId/reply
- Requires auth + venueId
- Stores reply text against comment
- Emails client if clientEmail was provided
- Returns updated comment

Register route in server/routes.ts
