CREATE server/routes/we-collab.ts
Register: app.use('/api/we/collab', weAuth, weVenue,
  requireCollaboration)

PAGE LOCK SYSTEM:

POST /api/we/collab/:projectId/:pageId/lock
- Records that userId is editing this page
- Stores in we_audit_log: action 'page_locked',
  metadata: { userId, pageId, lockedAt }
- Returns { locked: true, lockedBy, lockedAt }
- If already locked by different user:
  Returns { locked: false, lockedBy, lockedAt }
  Client shows: '[Name] is currently editing this page'

DELETE /api/we/collab/:projectId/:pageId/lock
- Releases lock for userId
- Only the user who locked can release
- Logs to we_audit_log: action 'page_unlocked'

GET /api/we/collab/:projectId/:pageId/lock
- Returns current lock status
- { isLocked, lockedBy, lockedAt, expiresAt }
- Lock expires after 30 minutes of inactivity

POST /api/we/collab/:projectId/:pageId/heartbeat
- Extends lock expiry by 30 minutes
- Called every 5 minutes while user is actively editing
- Returns { expiresAt }

GET /api/we/collab/:projectId/locks
- Returns all active page locks for project
- { locks: [{ pageId, lockedBy, lockedAt, expiresAt }] }

LOCK EXPIRY JOB:
- Create server/jobs/we-lock-expiry.ts
- Runs every 5 minutes
- Finds all locks older than 30 minutes in we_audit_log
- Releases expired locks
- Logs expired lock releases

CLIENT PREVIEW LINK SYSTEM:

POST /api/we/collab/:projectId/preview-link
- No requireCollaboration — available all tiers
- Generates unique preview token
- Stores in we_deployments with type: 'preview'
- Stores expiresAt: 7 days from now
- URL format: preview.indexflow.cloud/[token]
  if white label not configured
  preview.[agencyDomain]/[token]
  if white label configured
- Returns { previewUrl, expiresAt, token }

DELETE /api/we/collab/:projectId/preview-link/:token
- Immediately invalidates token
- Old link returns 404 after invalidation

GET /api/we/collab/:projectId/preview-link
- Returns current active preview link if exists
- Returns null if no active link or all expired
