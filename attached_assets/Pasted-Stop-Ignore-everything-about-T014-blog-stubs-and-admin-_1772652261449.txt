CREATE server/services/we-optimiser.ts

OPTIMISATION PIPELINE:
Takes raw HTML/CSS/JS and produces two output folders:
/src/ (readable) and /dist/ (optimised)

EXPORTS:

minifyHtml(html: string): string
- Remove comments, collapse whitespace
- Keep inline scripts and styles intact

minifyCss(css: string): string
- Remove comments, collapse whitespace

minifyJs(js: string): string
- Remove comments, collapse whitespace
- Defer all non-critical scripts

optimiseImage(filePath: string): Promise<OptimisedImage>
- Compress to 85% quality
- Max width 2000px — downscale if larger, never upscale
- Convert JPG and PNG to WebP
- Generate srcset variants: 400w, 800w, 1200w, 1600w
- Return { original, optimised, srcset[], savings }

inlineCriticalCss(html: string, css: string): string
- Extract above-fold CSS rules
- Inline in <head> as <style>
- Load full stylesheet deferred

generateSrcsets(html: string): string
- Find all <img> tags
- Add srcset and sizes attributes
- Add loading="lazy" to all images except first
- Add loading="eager" to first image (LCP target)

bundleFonts(html: string, projectId: string,
  venueId: string): Promise<string>
- Find all Google Fonts CDN links in html
- Download each font file
- Save to we_project_assets as font type
- Replace CDN links with self-hosted paths
- Add <link rel="preload"> for each font
- Return updated html

CREATE server/services/we-exporter.ts

EXPORTS:

generateExport(projectId, venueId): Promise<ExportJob>
- Validates tier — rejects with 403 if solo or trial
- Reads all pages from we_pages for project
- Reads all assets from we_project_assets
- For each page: generates clean HTML from grapejsState
- Assembles /src/ folder — readable unoptimised output
- Assembles /dist/ folder — runs full optimiser pipeline
- Generates sitemap.xml from all public pages
- Generates robots.txt
- Generates README.md in projectLanguage — includes:
  all page names and slugs, access levels, fonts used,
  third-party scripts, SSL setup guidance,
  form handler instructions for exported sites
- Generates optimisation-report.txt — before/after file
  sizes for all assets
- Bundles all into ZIP
- Stores ZIP path in we_deployments record
- Returns download URL expiring in 24 hours
- Entire process is async — returns job ID immediately

getExportStatus(jobId, venueId): Promise<ExportJobStatus>
- Returns { status, downloadUrl, expiresAt, error }
